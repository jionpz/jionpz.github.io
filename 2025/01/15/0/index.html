机器视觉OpenCV学习笔记

发表于2025-01-15|更新于2026-03-08|开发实践
|浏览量:

这篇笔记主要记录 OpenCV 学习初期的环境准备内容，包括 Python 环境、容器方案以及一些常用系统依赖。

当前内容还比较偏草记性质，后续可以继续补充版本选择、常见报错和示例工程结构。

这篇笔记记录什么

现阶段主要聚焦在“先把环境搭起来”，也就是把 Python 版本、容器环境和系统依赖准备好，确保后续样例代码可以顺利跑通。

等后续学习内容增多后，再继续补图像读取、视频处理、特征提取等更具体的 OpenCV 主题。

推荐起步方式

如果只是跟着入门教程走，比较稳妥的方式是先统一好 Python 版本，再补系统依赖，最后再安装 OpenCV 相关包，避免一开始就在环境差异上花太多时间。

建议先把环境跑通，再逐步加入自己的实验代码。

环境准备

配置 anaconda 环境，使用 .devcontainer 来创建环境。使用最新的 anaconda 环境后，需要切换 Python 版本至和视频一致。

anaconda 的一些基本命令

经验总结

OpenCV 入门阶段最容易卡住的往往不是算法本身，而是环境与依赖问题。

所以前期把 Python 版本、容器环境和系统依赖固定下来，通常能明显减少后续调试成本。
